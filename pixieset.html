Pixieset is a company that helps individuals start up their photogrphy business. It is An all-in-one platform for modern photographers, offering photo client galleries, websites, online stores and studio management software tools.
QA Engineers are expected to find bugs and abnormalities through manual testing and ensuring the correct data stored in the database. Often times QA Engineers will spend most of their time testing a task team's developer has worked on. Other key responsibilities for QA Engineers is writing test cases, generate team reports, and documenting important notes for a feature or testing scheme.

Software Engineer In Test

SET Engineers' main focus is to automate the manual test QA Engineers do. Often times this process is done by a javascript based automation framework, such as TestCafe. In Pixieset, we used the TestCafe framework which was a modular framework based on node.js. Main key responsibilities for SETs' are automating tests based on testcases, refactoring and maintaining the automation code base, and performing PR reviews for your fellow SET's (Very Important).
